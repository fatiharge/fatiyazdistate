
## How to use it?

wrap MyApp() in runApp

```dart
runApp(const FatihState(child: MyApp()));
```

Import it
Now in your Dart code, you can use:

```dart
import '../fati_yazdi_state.dart';
```

Usage
call from anywhere
[FatiYazdiState.fatihStateCreateAgain] FatihState.fatihStateCreateAgain(context);
